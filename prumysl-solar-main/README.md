# Prumysl - كاميرات المراقبة الشمسية

موقع إلكتروني لبيع كاميرات المراقبة الشمسية في المغرب.

## هيكل المشروع

```
prumysl-solar-main/
├── index.html              # الصفحة الرئيسية
├── moka.html              # صفحة منتج موكا
├── dio-moka.html          # صفحة منتج ديو موكا
├── assas.html             # صفحة منتج العساس
├── garde-corps.html       # صفحة منتج غارد كور
├── saqr.html              # صفحة منتج الصقر
├── about.html             # صفحة من نحن
├── contact.html           # صفحة اتصل بنا
├── faq.html               # الأسئلة الشائعة
├── return-policy.html     # سياسة الاسترجاع
├── terms.html             # شروط الاستخدام
├── thank-you.html         # صفحة شكر بعد الطلب
├── CNAME                  # ملف النطاق
├── css/
│   └── style.css          # ملف التنسيقات الرئيسي
├── js/                    # ملفات JavaScript (إن وجدت)
├── assets/                # ملفات إضافية
└── img/                   # الصور
    ├── logo.png
    ├── Desktop-hero.jpg
    ├── Mobile-hero.jpg
    └── [مجلدات المنتجات]/
```
